Llista d'exercicis del RACSO: 
- Regular descriptions(Reg)
- Context-free grammars(CFG)
- Deterministic Finite Automaton (DFA)
- Pushdown Automaton (PDA)
- Reduccions de K (K)
- Reduccions de CFG (CFG)
- Exàmens
